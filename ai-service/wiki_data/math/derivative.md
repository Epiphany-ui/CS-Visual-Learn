# 导数（Derivative）

## 一、定义
导数描述的是函数在某一点处的**瞬时变化率**，几何意义是函数曲线在该点切线的斜率。

数学上，函数 f(x) 在 x = a 处的导数定义为极限：
f'(a) = lim(Δx→0) [f(a + Δx) - f(a)] / Δx

如果这个极限存在，就称函数在该点可导。

## 二、核心原理
1. **割线逼近切线**：当两点无限靠近时，割线的斜率就趋近于切线的斜率，也就是导数
2. **变化率的度量**：导数为正 → 函数递增；导数为负 → 函数递减；导数为 0 → 极值点（驻点）
3. **二阶导数**：导数的导数，描述曲线的凹凸性。二阶导正 → 凹向上；二阶导负 → 凹向下

## 三、几何直观理解
想象一条曲线 y = f(x)：
- 在曲线上取两点 (x, f(x)) 和 (x + Δx, f(x + Δx))，连接两点得到一条割线
- 割线的斜率 = Δy / Δx = [f(x+Δx) - f(x)] / Δx
- 当 Δx 不断缩小，两点不断靠近，割线逐渐「贴到」曲线上
- 当 Δx 趋近于 0 时，割线变成切线，斜率就是该点的导数

## 四、常见求导公式
- 常数：(C)' = 0
- 幂函数：(xⁿ)' = n·xⁿ⁻¹
- 指数函数：(eˣ)' = eˣ
- 自然对数：(ln x)' = 1/x
- 正弦函数：(sin x)' = cos x
- 余弦函数：(cos x)' = -sin x

**求导法则**：
- 加法：(u + v)' = u' + v'
- 乘法：(u·v)' = u'v + uv'
- 除法：(u/v)' = (u'v - uv') / v²
- 链式法则：复合函数 f(g(x)) 的导数 = f'(g(x)) · g'(x)

## 五、常见误区
1. **可导一定连续，连续不一定可导**：比如 |x| 在 x=0 处连续但不可导（尖点）
2. **导数为 0 不一定是极值**：比如 f(x) = x³ 在 x=0 处导数为 0，但不是极值点，是拐点
3. **混淆导数与函数值大小**：导数大只代表变化快，不代表函数值大
4. **链式法则漏乘内层导数**：复合函数求导最常见的错误就是忘记乘以内层函数的导数
