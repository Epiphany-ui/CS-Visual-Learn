# 傅里叶变换（Fourier Transform）

## 一、定义
傅里叶变换是一种将时域信号转换为频域表示的数学工具。它告诉我们：**任何满足条件的非周期函数，都可以分解为无穷多个不同频率正弦波的叠加**。

连续傅里叶变换公式：
F(ω) = ∫(-∞到+∞) f(t) · e^(-iωt) dt

其中 f(t) 是时域函数，F(ω) 是频域函数，ω 是角频率。

## 二、核心原理
1. **时频对偶**：同一个信号，既可以用「幅度随时间变化」的时域视角看，也可以用「包含哪些频率分量」的频域视角看
2. **正交基分解**：不同频率的正弦波相互正交，傅里叶变换本质是信号在正交正弦基上的投影
3. **卷积定理**：时域卷积 = 频域乘积，这是信号处理、图像滤波的核心理论基础
4. **不确定性原理**：信号在时域越窄（持续时间短），在频域就越宽（频率分布广），反之亦然

## 三、与傅里叶级数的区别
- **傅里叶级数**：针对**周期信号**，分解为离散的基频整数倍频率分量，频谱是离散的谱线
- **傅里叶变换**：针对**非周期信号**，分解为连续的频率分量，频谱是连续的曲线

直观理解：当周期信号的周期 T 趋近于无穷大时，就变成了非周期信号，离散的谱线间距不断缩小，最终变成连续的频谱——这就是傅里叶变换。

## 四、典型时频对应对
1. **矩形脉冲 ↔ sinc 函数**
   - 时域：宽度为 τ 的矩形门函数
   - 频域：Sa(ωτ/2) 形状的 sinc 函数，主瓣宽度与脉宽成反比
2. **冲激函数 δ(t) ↔ 常数 1**
   - 时域无限窄 → 频域无限宽，均匀包含所有频率
3. **高斯函数 ↔ 高斯函数**
   - 时频域都是高斯形状，是不确定性原理取等号的极值情况
4. **正弦信号 ↔ 两个冲激**
   - 单频正弦波在频域表现为 ±ω₀ 处的两个冲激

## 五、常见误区
1. **混淆傅里叶级数与傅里叶变换**：级数是离散谱（周期信号），变换是连续谱（非周期信号），不能混用旋转圆圈的 epicycle 可视化来表示傅里叶变换
2. **以为只有周期信号才能分解**：非周期信号同样可以分解，只是频率分量是连续的，不是离散的几根谱线
3. **忽略相位信息**：傅里叶变换结果是复数，包含幅度谱和相位谱，只看幅度谱丢失了相位信息，无法完整还原信号
4. **采样与混淆**：离散傅里叶变换（DFT/FFT）必须满足奈奎斯特采样定理，采样率不足会产生频率混叠
